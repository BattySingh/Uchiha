Every file creates or defines one thing for our app.

Types of files:
1.) Component Template -> Contains the HTML that gets displayed to a user
2.) Component Class -> Contains code to run when important events occur
3.) Service -> Contains code that fetches/stores/updates data
4.) Module -> Defines a set of services + components that handle all the functionality for a part of our app

A single app usually has multiple modules, one for each major feature.
