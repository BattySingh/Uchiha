int myArr[5] = {1, 2, 3, 4, 5}; // declaring and assigning values inside of an array

int newAr[] = {1, 2, 3}; // if we do not specify size, the size of array becomes equal to the number of elements

for (a:newAr) { // to loop through all elements in an array. It is a range based for loop
  • cout << a;
}

For each loop is used with a collection of elements.

for (int x : newAr ) { // for each x in newAr, for each loop, x is different variable, and it gets copy of element, it is not holding original value

}

for (int &x : newAr) { // & is reference, used when we need to change value in the array using for each loop
  • x++;
}

auto -> keyword, if we do not know data type, automatically compiler will give type to variable according to value

Nested loops are used for Multi-dimensional arrays. Inner loop runs first.

2-D array: a[3][4] -> means array with 3 rows and 4 columns
