Enumerator Data type and typedef.

Enum is useful for defining user-defined data types.
In enum, the starting is always 0 (kind of index) or define yourself.

enum -> keyword

enum days { mon, tues, wed, thurs, fri, sat, sun };

int main() {
  • days day = wed; // this is how to use enums
}

enums are set of constants and you cannot change them once defined.


typedef -> Type Definition, to make program more readable
typedef -> keyword
ex: typedef int marks;

int main() {
  • marks m1, m2;
}


#define mon 0 -> these are symbolic constants
const int mon = 0; -> another way to define a constant
