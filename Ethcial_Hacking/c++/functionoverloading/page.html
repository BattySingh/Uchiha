Function Overloading is that we can write different functions with same name, but with different argument list.

Function Template: Functions that are Generic. Meaning functions which are generalized.

template <class T> // definition of template T. it is defined as T of class template. T is a template class

T max(T x, T y) {                 // here T is template, now this function works for both int and float, and we have to write only this template function once, not different for each int and float
  • if (x > y) return x;
  • else return y;
}

When using template function, make sure you add type with the function name
max<type>(a, b);

ex: max<int>(3, 2);
max<float>(3.2, 3.2);
