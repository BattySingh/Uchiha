Ponters are a type of variables in C++.
A pointer is a variable that is used to store the address of the data.

2 types of variables:
a.) Data variables: Used to store data of the variable
b.) Address variables: Used to store address of the data

int *p; // * is used to differentiate between data and address variables. declaration
p = &x;                               // & tells, store addess of data in variable x into p, initialization

*p -> is called de-referencing, meaning going to address and taking the data

When program is executing, memory is didvided into 3 sections: Heap, stack, code section
Pointers help access the heap area, normally program cannot. Pointers help access heap indirectly.

Files on a computer, network connection, keyboard and mouse, printers are acces via pointers.

Heap memory is accessed dynamically. When we want to allocate memory in heap, we allocate it using arrays.
int a[3] = {1, 2, 3};
int *p;
p = new int[3]; -> new is used to create array inside heap. When you say new, you get address in heap
Heap memory is not deleted automatically, it remains there as long as program is running.

delete []p;          // deallocating memory in heap. deleting array p in heap
p = NULL;           // making pointer as null, instead use nullptr
If youo do not free memory, it is called memory leak.
