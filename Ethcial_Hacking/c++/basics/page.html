HOW TEXT BECOMES EXECUTABLE

A compiler takes source code you can read and write, and transforms it into a different format.
 - The compiler gives you error messages if it can't compiler your code
 - The output is generally called an object file.

A large program is made up of several files, and each file is compiled, then the object files are linked together to create an executable file (exe)
 - the linker gives you error messages if it can't link the objects.

Finally you run the application.

3 types of time in c++ program: 1. Compile time       2. Run time    3. Link time
