A class defines the idea of an object, what it can hold and what functions it can be asked to perform.
An object is an instance of a class.
Functions inside a class are called member functions.

A string is a collection of characters.
#include <string> -> to use string class

for unicode, use wstring

string name; -> string is a type, and name is a variable that holds string value.
name.length() -> length is a member function and gives us the length of the name.
name.find("x") -> returns index of the character if "x" is found in name variable.
if "x" is not found, then we get returned 0 or garbage value.

name.substr(start substring from where) -> used to create substring out of name variable.

getline(cin, name); -> takes complete string until hit enter
