A class is a definition of user-defined type. A class pulls together related data.

Generally Member variables are private, and functions are public. Some classes need constructor. Name of the constructor function is same as class name. Constructors have no return type.

private: ->below it comes private members
public: -> public members come under it

to_string() method accepts a single integer and converts the integer value or other data type value into a string.
