Functions are block of code that you can use repeatedly.
All functions must be declared before they can be called, and all functions have a return type.

*************************************************************************************************
Two functions can have same name as long as compiler can tell them apart
Taking different number of arguments is a great way to distinguish overloads.
Return type cannot be used to distinguish overloads
Taking same number of arguments, but of different types is risky.

#include ->  it is called a directive.
Anything  that starts with # is an instruction to the preprocessor, a step that runs before the compiler.

#include "functions.h"

Header files contain declaration. Header file extensions: .h, .hpp
