This are the operations that are performed on the bits of the data.

& -> bitwise AND
| -> bitwise OR
^ -> XOR
~ -> NOT
<< -> left shift
>> -> right shift

Bitwise operators are used when creating device drivers, system programs, etc.

a > b && a > c -> gives true if both condition are true. If first is false, second condition is not checked, this is called short-circuit
a > b || a > c --> if first is true, second condition does not matter, it does not gets evaluated, called short-circuit.

Dynamic declaration: for (int i = 0; i<=10; i++) -> here i isused only inside block, making memory efficient
