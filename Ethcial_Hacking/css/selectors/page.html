Element selector: using elements: h1, p, li, etc. Selecting using elements

<link href="path to file" rel = "stylesheet" type="text/css"> -> to link css file to our webpage

Class Selectors: set equal style for elements withing  the same class.

Universal Selector: * selector to style all elements on the page

ID selectors: set style using ID

Attribute selectors: Set style to all elements with same attributes.

CSS is case-insensitive.

Inline styles have highest priority.

Cascading means multiple rules can apply to the same element.
And Specificity is used to to resolve conflicts arising from multiple rules.

Priority: inline styles > id > class == attribute selector > tag
