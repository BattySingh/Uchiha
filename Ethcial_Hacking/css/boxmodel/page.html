Box Model contains: Content, Padding, Boder and Margin.
Margin-collapsing: Margins of 2 elements collapses and the bigger margin wins.

margin: top right bottom left

By default when we set width and height, we are setting width and height of the content.
box-sizing: content-box; // it is deafult
box-sizing: border-box; // to include the border inside the height. But margin cannot be included with the height.
