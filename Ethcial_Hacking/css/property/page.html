color: green; -> to change color of the text

font-family: sans-serif; -> to change the font-family of the text.

sans-serif -> uses browsers default font-family

font-family: inherit; inherit is a keyword, uses the parent's value.

Combinators allows us to select and combine selectors together.
ex: #id h1 { } -> select h1 inside id element

background: red; // to change the background color
background: url('path to image'); // to set a background image

background-repeat: no-repeat; // to make no repeat of background image
background-size: cover; // to cover image with whole body

border-color: red; // to set border color
border-width: 3px; // to give width to border
border-style: solid; // the type of border
