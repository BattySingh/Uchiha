1.) Adjacent combinator -> Elements share same parent. Second element comes immediately after first element.
ex: div + p {

}

2.) Sibling combinator -> Elements share same parent. Second element comes after first element, but does not have to be adjacent.
ex: div ~ p {

}

3.) Child combinator -> Second element is the direct child of first element.
ex: div > p {

}

4.) Descendant combinator -> Second element is the descendant of the first element.
ex: div p {

}
