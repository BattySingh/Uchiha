Windows have a command line utility called netsh (Network Shell), that allows you to display and moidy network configurations of your system.
The most common use of netsh is to reset the TCP/IP stack back to default. Use netsh from cmd using administrative privileges.

netsh advfirewall reset -> Restores the Windows Firewall with Advanced Security policy to default policy, this command returns all settings to not configured and deletes all connection security and firewall rules.

netsh int ip reset -> Removes all user configured IPv4 settings. Restarting of computer is required before the default settings will take effect.

netsh int ipv6 reset -> Removes all user configured IPv6 settings. Restarting of computer is required before the deafult settings take effect.

netsh winsock reset  -> Resets Winsock Catalog to a clean state. All Winsock layered Service Providers which were previously installed must be reinstalled. This command does not affect Winsock Name Space Provider entries.
