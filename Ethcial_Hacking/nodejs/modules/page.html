http -> node module to launch a server, and send requests.

https -> launch a SSL Server.

Creating a NodeJs server:
const http = require('http'); // getting the core module into our file
const server = http.createServer((req, res) => {
  • console.log(req); // req or request is an object
});

server.listen(3000); // listen method, listens requests on port 3000

Entire Node app uses one single thread.

process.exit() -> to quit our server.

Headers are meta data added to our requests.

req.url -> to get the url
req.method
req.headers -> to get headers information

NodeJs is Event Driven. It responds whenever an event occurs.
