Node.js file system module allows you to work with the file system on your computer.

fs ( File System )

var fs = require('fs'); // to include the file system into your application

File system module can be used to : read files, create files, update files, delete files, rename files.

fs.readFile(); // is used to read files from your computer

ex: It is non-blocking way to read from a file
const fs = require('fs');
fs.readFile('fileName', 'utf-8', (err, data) => { // utf-8 is encoding of the file, it is important, if you do not add it, you will be displayed with a raw buffer
  • if (err) throw err; // if error is there in the file
  • console.log(data); // displays the contents of the file
});


Blocking way:
const data = fs.fileReadSync('fileName', 'utf-8');
console.log(data);

console.clear(); // clears the screen
