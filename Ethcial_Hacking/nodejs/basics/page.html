What is NodeJs?
A Javascript Runtime. It is Javascript on the server.

NodeJs used V8 engine, that runs javascript.
V8 takes the Javascript code and compiles it into Machine Code.
V8 is written in C++.

node -v -> gives us the version of node js installed on your system.

Type node in your terminal, and it will open node, that is called repl (Read Evaluate Print Eval)
Read -> read user input
Evaluate -> Evaluate user input
Print -> Print output (result)
Loop -> Wait for New Input

const fs = require('fs'); // here we are importing a file into our script
fs.writeFileSync('hello.txt', 'some text'); // this will create a "hello.txt" file in same folder and write the text "some text" into the file.

Alternatives to nodeJs: python, php, asp.net, ruby on rails.

HTTP -> Hyper Text Transfer Protocol
A protocol for tranfering Data which is understood by Browser and Server.

HTTPS -> Hyper Text Transfer Protocol Secure
HTTP + Data Encryption (during Transmission)


Node.js is a Javascript runtime built on Chrome's V8 Javascript engine.  Node.js uses an event-driven, non-blocking model that makes it lightweight and efficient. Node.js package ecosystem, npm, is the largest ecosytem of open source libraries in the world.

NodeJs actually is a platform which allows us to run Javascript on a computer/server. It helps us read, delete and update files. It helps to easily communicate with the database.
