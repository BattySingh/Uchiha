pip install django -> to instal Django on your system

python -m django --version -> gives us the version of django installed on our system

Django is a web development framework. It is a backend framework.

django-admin -> gives django commands, it's a utility
django-admin startproject mysite -> to create a mysite folder. mysite is project name

manage.py is utitlity, that helps manage our project
__init__.py -> package file
settings.py -> contains settings for our project
urls.py -> conatins url declarations
wsgi.py -> WSGI (Web Server Gateway Interface)

python manage.py runserver -> starts development server
