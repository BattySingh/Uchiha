Powershell is a Management Engine.
It is a dll basically.
It is built on .NET framework.
Hosted by an application.

One is terminal console (available everywhere), and second is Graphical Console ( called Powershell ISE (Integrated Scripting Editor), it is by default installed on Windows but not on Windows server )

$psversiontable -> gives us the information about powersehll, like it's version number, edition, serialization version.

exit -> to get or quit the powershell

ise -> opens up powershell ise from powershell

powershell -> in cmd opens the powershell inside of the cmd window

To get help in powershell: help commandName
ex: help get-service -> gives us few detail on how to get help on using with the command

update-help -> downloads help files into your system from internet. The help files are stored in system32, therefore you need to have elevated priveleges, administrator powershell.
Updates the documentation

update-help -force -> -force is used to tell powershell, download it again completely.

get-service -name bits, lanmanserver -> to get names of services, to check if they are running or not

get-service bits, winrm -computername computreNameHere -> to check for that computer

cls
clear -> both clear the screens

help get-service -examples // to get only examples

help get-eventlog -parameter logname -> narrowing search on what to find

help get-service -online -> opens help documentation online
help get-service -showwindow -> opens help in a window outside console

help *service* -> using wildcards

help about* -> gives us about pages, you won't get these until you update help
