Powershell is an execution engine, that provides the ability for you to interface with your environment using different tools.

Powershell core is newer version of powershell, that helps run powershell commands on different environments.
Windows powershell only runs on Windows, and is feature complete.
It is the Powershell Core that runs on all Operating systems, like windows, linux and Mac. It is open-source.

Why use platform? Because: 1.) It is baked into Windows 2.) Automate tasks 3.) Everything under the hood uses powershell, so take more control.

Active Directory is a directory service developed by Microsoft for Windows Domain networks.

get-service // command gives us all the services on our system, running and stopped
get-service | // | is a pipe, and when hit enter, cmd waits for us to enter a command
where-Object Status -eq 'Stopped' | // where-object is used to get a more specific, of what kind of resulting service we need, like here we ask to display all stopped services.
select-object Name, Status // to display only name and status of stopped services

$data = get-service | where-object status -eq 'stopped' | select-object name, status // here we are storing result inside $data variable

Now, we can use this variable to create a .csv file and store result inside it.
$data | out-file .\ex.csv // stores the result of $data into a csv file

$data | export-csv .\ex1.csv // this will format each column into it's new column in excel, separating the values by quotes and commas
get-content .\ex1.csv // to view the content of file inside the console.
