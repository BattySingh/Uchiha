"Remote origin already exists" error. -> this shows up when the origin is already set, and to solve it you have set the link to new using below command

git remote set-url origin linkComesHere

When you get error of accessd denied to user that isn't you and you don't get asked your credentials, is because windows has saved the credentials and it is trying to use those old credentials. To resolve the error, you could perform below steps:

control panel > user accounts > credential manager > windows credentials > remove git credentials from the list
