If you do not write code inside function, that means you are writing code in global scope.

A function is a block of code with a name.

syntax: function functionName() {
                     // block of code here
  •     }
"function" is a keyword.

Another way to declare a function is "Function Expression".

syntax for Function expression:

let x = function() {
  • // code comes here
}

x(); // calling the function

FUNTIONS have access to entire outter scope.

"document" is an object that refers to the whole page.

document.getElementById('idName').textContent.

getElementById -> to target the element using ID
textContent -> is used to access the content, meaning what is there at provided ID
