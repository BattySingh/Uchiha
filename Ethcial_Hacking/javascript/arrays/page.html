Arrays are objects that can hold multiple values or objects.

let ar = []; // creating an array

Each value in an array is called an element.

Another way to create an array:
let val = Array.of(1, 2, 3);

In arrays, we start counting from 0.

Array.isArray(variable); // to check if a variable is an Array or not

variable.push('value') // adds or pushes element at end of array, can push more than one values
variable.pop() // takes out the element from end of array

variable.shift() // shifts an entire array, and removes the element from front of array
variable.unshift('value') // adds value to start of the array

slice -> used to create a new arrau, based of an original array
splice -> is used to insert or delete element from an array at any position.

variable.slice(startIndex, endIndex); // slicing an array
variable.splice(indexOfElementToStartDeletingFrom, numberOfElementsToDelete);
variable.splice(indexOfElementToStartDeletingFrom, numberOfElementsToDelete, 'valueToInsertAfterDeleting');

variable.indexOf('element') -> gives us the index value of an element, returns -1, if element not present in the array

variable.filter(function(item) { // loops through an array, and can be used to check for a condition and return those elements
  • return item;
})

variable.find(function(item) { // similar to filter function, but returns the first find value
  • return item;
});
