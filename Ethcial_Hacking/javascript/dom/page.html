A webpage is mainly created using a document object.

document.getElementById('idName').addEventListener('NameOfEvent', 'annoymousFunction'); // to listen to different types of events

document.getElementById('idName').classList.remove('className'); // classList to list all classes, and remove to remove specified class, toggle to on/off alternatively, contains let us know if a certain class is present in defined element or not

document.getElementByClassName('className') // to target element based in class name

d-none -> it is a bootstrap class, stands for 'Display None'.

Javascript is backwards compatible with it's earlier versions.

Document object is created by the browser and models the HTML page.
document.URL // gives the url of current page

document.querySelector('p'); // only selects the first p tag, and igonores all other p tags
document.querySelector('.className'); // to target a class name element
document.querySelector('div.test'); // to target div with className

document.querySelectorAll('p'); // to target all p tags, not just first element, gives nodeList, so dot notation could be used to get elements, forEach method could be used
document.getElementById('idName'); // to target elements by id
document.getElementsByClassName('className'); to target using class names, gives HTML collection, cannot use forEach method in this
document.getElementsByTagName('p'); to target elements using tags, it also gives an HTML collection
