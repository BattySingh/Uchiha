Math -> math object

Math.PI -> gives value of pi
Math.E

Math.round(3.3) // rounds to the nearest integer
Math.floor(3.3)  // gives the lower integer value
Math.ceil(3.3) // it is opposite of floor
Math.trunc(3.3) // removes the decimal part
Math.random() // used to generate random number between 0 and 1.

Primitive types: numbers, strings, booleans, null, undefined, symbol
Reference type: all types of objects: object literals, arrays, functions, dates, and other objects.

In javascript, object created are stored inside the heap, whereas primitive data is stored inside stack(but has limited area).
