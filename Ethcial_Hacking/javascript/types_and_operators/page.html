Spread operator is ... (3 periods). It is used to add the elements of an existing array into a new array. It can be used to pass the elements of the array as arguments to the function. It can be used to copy arrays. It can be used to concatenate arrays.

Rest operator: It is like the spread operator, it condense multiple elements into an array.

Destructuring Assignment is a Javascript syntax that allows us to extract data from arrays and objects into distinct variables. It is used to assign variables from objects. Used to assign variabes from nested objects. Also used to assign variables from arrays. It is used to reassign array elements. It can be used to pass an object as a functions parameters.
Ex: var vowel = {x: 3.6, y: 7.4, z: 6.54};
const {x, y, z} = vowel; -> this creates 3 variables x, y, z, and stores vowel values into each variable.
or const {x: a, y: b, z: c} = vowel; -> copy value of x into a, etc.
