Arrays are created using: val myArr = arrayOf("one", "two", "three");

myArr.size -> to get the size of the array
myArr[1] -> to access individual array values

myArr.get(1) -> works as index, gets the value of element at index1 , index starts from 0.

We can mix types inside of array.

For loop to iterate all elements in an array,
for (num in myArr) {
  • println(num);
}

also used for looping through an array
myArr.forEach { -> traverses through an array
  • println(it); -> it is deafult name for elemenst in an array, and prints all elements one by one from array
}

if we want to change variable name "it" to our custom name, use
myArr.forEach { item ->                                                      // so we can use item as name for array elements, lambda syntax
  • println(item);
}
