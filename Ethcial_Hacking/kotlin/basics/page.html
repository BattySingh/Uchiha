Kotlin is a statically-typed programming language, developed by Jetbrains. It's around 2011. It is primary language for Android app development.

There should always be a main function in your kotlin file.

fun main() { // fun is a keyword used to declare and define a function, with main as a function
  • println("hello"); // println is used to output message onto the screen.
}

print -> function can also be used to display message onto the console screen
And always use "" -> double quotes to output string, single quotes causes error.

There are 2 types of variables in kotlin: 1) Mutable (whose value can be reassigned)      defined using "var" keyword
2.) Local desfined read only variables (who have value assigned to them only once)       defined using "val" keuword


val variableName: variableType = variableValue;
Ex: val name: String = "Goku";

If we try to re-assign value to the val variable, compiler gives us an error.

// -> is for single line commenting code

Difference between println() and print()  is that println() adds enters or let's say it takes cursor to the new line, while print() does not, and if we write text using print(), text just gets entered in horizontal.

In kotlin, we can define variables outside of functions or classes, making them global variables, and they are called "Top Level Variables".
