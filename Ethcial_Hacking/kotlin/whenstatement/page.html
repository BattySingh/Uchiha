when statement is similar to switch statement in java. Used to manipulate control flow just like if-else statements.

when is keyword
when (condition or expression) {
  • null -> println(); // -> follows what action to take, if null
  • else -> run this
}

When two different packages each use the same name, then we can use the "as" keyword to alias the name. This is especially useful when common names are used by multiple libraries.

import com.packt.myproject.Foo
import com.packt.otherproject.Foo as Foo2

println("Hello ${5 + 4}"); // brackets can be used want to use expressions.

A range is defined as an interval that has a start value and an end value. Any types which are comparable can be used to create a range, which is done using .. operator.
val atz = "a".."z"

downTo() // will create a range counting down
ex: 100.downTo(0)

rangeTo() // will create a range up to a value.
ex: 10.rangeTo(20)
