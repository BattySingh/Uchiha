listOf() -> is used to create a list, and like arrays it can also contain different types of values into it, like string, integers, booleans.

Unlike arrays, that on prinltn(arrayName), gives memory location where array is stored, In list, println(listName); gives the whole list.

mapOf(); -> is used to create map, which is a key value pair.
ex: mapOf(1 to "a", 2 to "b", 3 to "c");

Arrays, Lists, Maps are collections.

forEachIndexed does not works with Map collections.
