we can run c# on top of runtime and framework .NET Core. Also .NET Core SDK(Software Development Kit)

2 types of .NET:
1. .NET Framework: only for windows.
2. .NET Core is open source, and works on Windows, Linux and Mac.

.NET is the software that converts our C# code to machine understable language.
.NET has 2 main parts:
    1. CLR (Common Language Runtime)
    2. FCL (Framework Class Library)

dotnet -> command line tool

dotnet --info -> Provides informationo about .net core version, operating system name, version, platform, base path, etc.

ASP.NET -> it is a web programming framework

dotnet new console -> to create a new console application project

.csproj -> this extension means it is a c# project
.cs -> extension for C# files.

dotnet run -> to run our c# project.

dotnet run --project src/folderName

dotnet restore -> command to grab dependies

in dotnet, there is a package system called 'nuget', it helps my application get advantage of code that other people have written.

dotnet build -> to analyze my code

here it happens, that my all project files combine together to form a new binary file after compilation, .dll file.

"obj" folder contains temporary files that are created using restore and build
dotnet x.dll -> to run dll files.
